# Screenshot captions

Use these captions in GitHub README, portfolio posts, or LinkedIn project descriptions.

1. **Dashboard Overview** - shows document counts, review status, validation status, and recent workflow activity.
2. **Processed Document List** - displays seeded demo documents with processing status and document type information.
3. **Document Detail and Validation** - shows extracted fields, classification output, and validation result for a processed document.
4. **Human Review and Edit Warning** - demonstrates the human-in-the-loop correction workflow before approval.
5. **Validation Results** - displays business-rule validation issues that block or warn before approval.
6. **Export Page** - shows approved-only CSV/JSON export controls.
7. **Audit Log** - shows the traceable event history for document upload, extraction, validation, review, and export.
8. **Demo Mode Seeded Documents** - shows the zero-credential demo mode that loads sample invoices, receipts, purchase orders, and contracts.
9. **FastAPI Swagger Documentation** - shows the backend API endpoints available at `/docs`.
