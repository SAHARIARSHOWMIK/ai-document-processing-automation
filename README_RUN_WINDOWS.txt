AI Document Processing Automation - Windows Run Guide
====================================================

Use Python 3.12.

First-time setup:
1. Extract the zip.
2. Open the project folder.
3. Double-click setup_windows.bat.

Run the app:
1. Double-click start_backend.bat.
2. Double-click start_dashboard.bat.
3. Open http://localhost:8501.

API docs:
http://localhost:8000/docs

Demo flow:
1. Open Dashboard.
2. Go to Demo Mode.
3. Click Load Sample Documents.
4. Open Documents and process/review a sample document.
5. Export approved records.
6. Check Audit Log.
